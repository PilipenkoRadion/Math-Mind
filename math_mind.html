🧮 MathMind Flash Anzan v3.5

Gesture-Controlled Math Game Platform

⚡ FastAPI Backend

Komponente Beschreibung
FastAPI Hochperformanter Web-Server mit CORS-Middleware.
Uvicorn ASGI-Server für FastAPI auf Port 5000.
Python 3.8+ Runtime für Backend-Logik.
Threading Parallele Verarbeitung für Hand-Erkennung und Server.
JSON Datenaustausch zwischen Frontend und Backend.

🤖 MediaPipe Hand Detection

Komponente Spezifikation
HandLandmarker 21 Landmarks pro Hand, detektiert bis zu 2 Hände gleichzeitig.
Finger Counting Zählt 0-5 Finger durch Landmark-Analyse.
GestureStabilizer Stabilisiert Gesten mit 2.5s Lock-Zeit und CONFIRM-Schwellen.
Confidence Threshold Min. 0.55 für Hand-Detection und Presence.

📷 OpenCV & Video Processing

Feature Details
Video Capture 640×480 @ 30 FPS mit Buffer-Size = 1.
Camera Access DirectShow (Windows) oder Standard-Device.
Frame Flip Horizontal flippen für Spiegelbild-Effekt.
RGB Conversion BGR → RGB für MediaPipe-Kompatibilität.
Fallback Finger Count HSV-basierte Hautfarb-Detektion bei MediaPipe-Fehler.

🎨 Frontend (HTML/CSS/JS)

Technologie Zweck
HTML5 Struktur der Game-UI (Menu, Flash, Schwer Screens).
CSS3 Gradient-Design, Animationen, Responsive Layout.
Vanilla JS Game-Logik, State-Management, Event-Handling.
Fetch API Kommunikation mit FastAPI Backend.
Countdown 3-2-1 Overlay mit Animation vor Spielstart.

✨ Game Features

🎮 Flash Mode

Schnelle Zahlenfolgen (Leicht: 5 Zahlen, Mittel: 8 Zahlen). Spieler berechnet Summe.

📚 Schwer Mode

3 Schwierigkeitsstufen (A/B/C): Math-Aufgaben, Mixed Ops, Zahlenreihen & Textaufgaben.

🖐️ Gesture Control

1=Leicht, 2=Mittel, 3=Schwer, 4=Neustart, 5=Beenden (Finger-Gesten).

📊 Score Tracking

Visuelle Punkte-Anzeige (grün/rot) für jede Aufgabe mit Accuracy-Bars.

⏱️ Timing

Countdown vor Spielstart, Gesture-Lock 2.5s, Stabilization (CONFIRM_FAST/SLOW).

🎯 Task Generations

Zufallsaufgaben: gen_task_a/b/c mit Fibonacci-Sequenzen & Wort-Probleme.

🔄 Systemarchitektur

📷 Webcam (640×480 @ 30 FPS) ↓ 🐍 Python Backend (FastAPI) ├─ OpenCV: Frame Capture & RGB Conversion ├─ MediaPipe: Hand Landmarks (21pt × 2 hands) ├─ Finger Counter: 0-5 Finger Detection ├─ GestureStabilizer: Stabilisierung + Lock-Zeit └─ GameState: Flash/Schwer Game Logic ↓ REST API Endpoints ├─ /api/status → Camera Status, Finger Count ├─ /api/fingers → Aktuelle Finger ├─ /api/gesture → Stabile Gesten ├─ /api/start/{level} → Flash Game Start ├─ /api/check/{answer} → Antwort Validierung ├─ /api/schwer/start/{sub} → Schwer Mode Start ├─ /api/schwer/check/{answer} → Task Check ├─ /api/schwer/next → Nächste Aufgabe ├─ /api/restart → Neustart Game └─ /api/exit → Menu Return ↓ 🌐 Frontend (Browser) ├─ HTML5: Menu, Flash Screen, Schwer Screen ├─ CSS3: Gradient, Animations, Responsive ├─ JS: Game State, Fetch Calls, UI Updates ├─ Countdown Overlay: 3-2-1-Bereit? └─ Finger Counter Display: Real-time
