Spielfunktionen

Flash-Modus

Schnelle Zahlenfolgen (Leicht: 5 Zahlen, Mittel: 8 Zahlen). Spieler berechnet die Summe.

Schwer-Modus

3 Schwierigkeitsstufen (A/B/C): Mathematik, Gemischte Operationen, Sequenzen & Textaufgaben

Gesterenkennung

1=Leicht, 2=Mittel, 3=Schwer, 4=Neustart, 5=Beenden (Finger-Gesten)

Punktverfolgung

Visuelle Punkteanzeige (grün/rot) für jede Aufgabe mit Genauigkeitsbalken

Timing

Countdown vor dem Spiel, Gestensperrung 2,5s, Stabilisierung

Aufgabenerzeugung

Zufallsaufgaben: gen_task_a/b/c mit Fibonacci & Textproblemen

Systemarchitektur

EINGABESCHICHT:
📷 Webcam (640×480 @ 30 FPS)
↓
VERARBEITUNGSSCHICHT (Python Backend):
├─ 🎥 OpenCV: Frame Erfassung & RGB Umwandlung
├─ 🧠 MediaPipe: Hand Landmarks (21pt × 2 Hände)
├─ 🔢 Finger Zähler: 0-5 Erkennung
├─ 🔒 GestureStabilizer: Sperrzeit & Stabilisierung
└─ 🎮 GameState: Flash/Schwer Spiellogik
↓
REST API ENDPUNKTE:
├─ /api/status → Kamerastatus, Fingerzahl
├─ /api/fingers → Aktuelle Finger
├─ /api/gesture → Stabile Gesten
├─ /api/start/{level} → Flash Spiel Start
├─ /api/check/{answer} → Antwort Validierung
├─ /api/schwer/start/{sub} → Schwer Modus Start
├─ /api/schwer/check/{answer} → Aufgaben Kontrolle
├─ /api/schwer/next → Nächste Aufgabe
├─ /api/restart → Spiel Neustart
└─ /api/exit → Menü Rückkehr
↓
AUSGABESCHICHT (Frontend):
🌐 HTML5: Menü, Flash-Bildschirm, Schwer-Bildschirm
🎨 CSS3: Verlauf, Animationen, Responsive
📊 JavaScript: Spielzustand, Abruf, UI-Updates
⏱️ Countdown Overlay: 3-2-1 Animation
👆 Fingerzähler Anzeige: Echtzeit

Datenfluss: Webcam → OpenCV/MediaPipe → Verarbeitung → FastAPI REST/WebSocket → Frontend JavaScript
