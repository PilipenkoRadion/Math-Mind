Game Features

Flash Mode

Быстрые числовые последовательности (Легко: 5 чисел, Средне: 8 чисел). Игрок считает сумму.

Schwer Mode

3 уровня сложности (A/B/C): Математика, Смешанные операции, Последовательности & Текстовые задачи

Gesture Control

1=Легко, 2=Средне, 3=Сложно, 4=Перезагрузка, 5=Выход (Палец-жесты)

Score Tracking

Визуальное отображение баллов (зеленый/красный) для каждого задания с точностью

Timing

Обратный отсчет перед игрой, блокировка жеста 2.5s, стабилизация

Task Generation

Случайные задачи: gen_task_a/b/c с Фибоначчи & Текстовыми проблемами

Системная архитектура

INPUT LAYER:
📷 Webcam (640×480 @ 30 FPS)
↓
PROCESSING LAYER (Python Backend):
├─ 🎥 OpenCV: Frame Capture & RGB Conversion
├─ 🧠 MediaPipe: Hand Landmarks (21pt × 2 hands)
├─ 🔢 Finger Counter: 0-5 Detection
├─ 🔒 GestureStabilizer: Lock-время & Stabilization
└─ 🎮 GameState: Flash/Schwer Game Logic
↓
REST API ENDPOINTS:
├─ /api/status → Camera Status, Finger Count
├─ /api/fingers → Current Fingers
├─ /api/gesture → Stable Gestures
├─ /api/start/{level} → Flash Game Start
├─ /api/check/{answer} → Answer Validation
├─ /api/schwer/start/{sub} → Schwer Mode Start
├─ /api/schwer/check/{answer} → Task Check
├─ /api/schwer/next → Next Task
├─ /api/restart → Game Restart
└─ /api/exit → Menu Return
↓
OUTPUT LAYER (Frontend):
🌐 HTML5: Menu, Flash Screen, Schwer Screen
🎨 CSS3: Gradient, Animations, Responsive
📊 JavaScript: Game State, Fetch, UI Updates
⏱️ Countdown Overlay: 3-2-1 Animation
👆 Finger Counter Display: Real-time

Поток данных: Webcam → OpenCV/MediaPipe → Processing → FastAPI REST/WebSocket → Frontend JavaScript
